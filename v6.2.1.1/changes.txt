VNM SimCenter v6.2.1.1 released

### New features
 * VNM Sim Center MCP is now available at /mcp endpoint (experimental). You can add it to your favourite AI agent to control the app. Current available tools:
    * getControlList: Returns a list of all registered logical control IDs and their display labels
    * getAllControlStatus: Returns the current labels and values for all registered controls (excluding buttons)
    * getControlState: Retrieves the current value, type, items (if applicable), and supported actions for a control ID
    * executeAction: Performs an action (set, inc, dec, click, select) on a specific control
    * more tools will be added in the future

Here is a sample Antigravity config (mcp_config.json):
{
  "mcpServers": {
    "vnm-simcenter": {
      "url": "http://localhost:900-/mcp"
    }
  }
}
